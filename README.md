# BokplasseringDeichman

En app for å legge inn ider på bøker som er på Deichman Bjørvika og få en liste over hvor de er plassert.

Søker bare på Bjørvika. Dette kan man enkelt fikse på.


## Installasjon

Testet med 
- Android devtools 26
- node 16.15.1
- npm 8.5.5
